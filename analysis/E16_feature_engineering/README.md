# E16: Feature Engineering

Extracts 17 statistical and temporal features from RSSI time series.

## Features Extracted
- Statistical: mean, std, min, max, median, range, quartiles, IQR
- Distribution: skewness, kurtosis, coefficient of variation
- Temporal: rate of change, max change

## Files
- `engineered_features.csv` - All extracted features
- `feature_correlations.csv` - Feature correlations with distance
- `feature_engineering.png` - Feature analysis plots
